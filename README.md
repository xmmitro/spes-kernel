# ⚡ KSU Next Kernel for Spes (Redmi Note 11 4G)

Custom Linux kernel for **Redmi Note 11 4G (spes/spesn)**  
Focused on **KernelSU Next + Docker support + performance**

---

## 📱 Device Info
- Device: Redmi Note 11 4G
- Codename: spes / spesn
- Chipset: Snapdragon 680 (SM6225)

---

## 🚀 Features
- ✅ KernelSU Next support
- 🐳 Docker support (enabled in kernel)
- ⚡ Performance & stability improvements
- 🔧 Based on Xiaomi OSS kernel source
- 🔒 Root-friendly (no need to reflash Magisk in most cases)

## 🛠️ Installation

### Requirements:
- Unlocked bootloader
- Custom recovery (TWRP / OrangeFox)

### Steps:
1. Reboot to recovery
2. Flash the kernel zip
3. Reboot system

⚠️ **Always take a backup before flashing**

- ROM: Evolution X (tested)
